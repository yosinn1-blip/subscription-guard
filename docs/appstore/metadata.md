## 説明文（4000文字以内）

```
無料トライアルを登録したまま、うっかり課金されていませんか？

Subscription Guard は「今トライアル中のサービス」だけを管理し、終了前日に通知する、それだけのアプリです。

━━━━━━━━━━━━
使い方はシンプル
━━━━━━━━━━━━
① 無料トライアルを申し込んだサービスを登録
② 終了日をセット
③ 前日に通知が届く → 解約するか継続するか判断する

━━━━━━━━━━━━
あえて機能を絞っています
━━━━━━━━━━━━
• 管理画面なし
• グラフなし
• 支払い合計なし

全サブスクを「管理」するアプリではありません。
「うっかり課金」を防ぐことだけに特化しています。

━━━━━━━━━━━━
対応サービス（25種類以上）
━━━━━━━━━━━━
Netflix / Spotify / Disney+ / Amazon Prime / YouTube Premium /
Apple Music / Apple TV+ / Hulu / U-NEXT / Crunchyroll /
iCloud+ / Dropbox / PlayStation Plus / Nintendo Switch Online /
Microsoft 365 / Notion / GitHub / Zoom / Discord Nitro /
ニコニコプレミアム / Twitch / Audible / LINE MUSIC / DAZN
… その他カスタム登録も可能

━━━━━━━━━━━━
プライバシー
━━━━━━━━━━━━
データはすべてデバイス内に保存されます。
外部サーバーへの送信は一切ありません。
```
